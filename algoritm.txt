1.1. Инициализация генератора случайных чисел
•	Функция srand(time(0))
1.2. Подготовка параметров для создания файлов
•	Создается массив строк filenames[], содержащий имена трех файлов: "file1.txt", "file2.txt", "file3.txt"
•	Задаются константы: num_files = 3 (количество файлов) и num_numbers = 10 (количество чисел в каждом файле)
1.3. Создание и заполнение файлов случайными числами
•	Запускается внешний цикл по количеству файлов (от i = 0 до i < num_files)
•	Для каждого файла:
o	Открывается поток записи ofstream с соответствующим именем из массива
o	Проверяется успешность открытия файла: если !file.is_open(), выводится сообщение об ошибке и программа завершается с кодом 1
o	Запускается внутренний цикл по количеству чисел (от j = 0 до j < num_numbers)
o	На каждой итерации генерируется случайное число от 1 до 10 по формуле: rand() % 10 + 1
o	Сгенерированное число записывается в текущий файл
o	Если число не последнее в последовательности (j < num_numbers - 1), после него добавляется пробел-разделитель
•	После заполнения каждого файла числами, файл закрывается методом file.close()
•	Выводится сообщение об успешном создании текущего файла
1.4. Завершение работы программы
•	После обработки всех трех файлов выводится итоговое сообщение "Все файлы успешно созданы!"
•	Программа завершается с кодом возврата 0, указывающим на успешное выполнение
